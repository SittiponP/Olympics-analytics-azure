# Olympics-analytics-azure

### Project Overview
in this project we are aims for managing, streamline Using Azure( Microsoft Could )

### Services we be using
1. Data Factory: use for build a data pipeline to load the csv file into azure data lake gen2
2. Data Lake Gen 2: for storing data here
3. Azure Databricks: use for transforming data and upload data back to data storage(data lake gen 2)
4. Azure Synapse Analytics: user for doing some analytics on it 
   
### Dataset Used
the dataset is from kaggle data contains 5 csv files Athletes, Coaches, EntriesGender, Medals, Teams
